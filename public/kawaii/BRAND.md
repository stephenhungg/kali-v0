# kali kawaii brand lock

## identity

- product: kali, an agentic context layer for nonprofits.
- promise: calm, cute, and soft on the surface; serious backend coordination underneath.
- mascot: a matcha cup/context-cafe character. fresh identity only, not the old strawberry-hat logo.
- wordmark: user-approved chunky rounded `kali` sticker lettering with white keyline, matcha fill, soft offset shadow, katakana `カリ`, and a context-cafe label.

## palette

- paper: `#fffdf6`
- cloud: `#ffffff`
- soft mint: `#f1fae9`
- matcha mid: `#78a86b`
- matcha deep: `#204c37`
- espresso ink: `#2d241c`
- cream foam: `#fff4df`
- sakura accent: `#ffd9de`
- yuzu accent: `#ffe7a3`

## recurring props

- receipt stickers

- approval stamps
- matcha cafe labels
- mint striped wrappers

## typography

- wordmark/display: chunky rounded sticker lettering.
- subtext/body copy: `Nunito`, because it is rounder and warmer than the old inter/swiss support copy.
- dense product emphasis: swiss display only when the interface needs sharper hierarchy.

## copy voice

- use sentence case for product copy, labels, buttons, and helper text.
- keep `kali` lowercase only when it is the literal logo/wordmark.

## higgsfield-ready prompt

asset type: transparent sticker logo and mascot sheet for a web app brand.

brand facts: `kali`, a cute but powerful agentic context layer for nonprofits. mascot is a smiling matcha latte cup/context-cafe character with cream foam, tiny leaf accent, rounded face, thick white sticker outline, matcha green body, espresso ink details, soft sakura cheeks, and tiny citation sparkle props. wordmark text must be exactly `kali` in readable chunky rounded roman letters.

palette: paper `#fffdf6`, cloud `#ffffff`, soft mint `#f1fae9`, matcha `#78a86b`, deep matcha `#204c37`, cream `#fff4df`, sakura `#ffd9de`, yuzu `#ffe7a3`, espresso ink `#2d241c`.

treatment: thick white keyline, soft offset matcha shadow, simple vector-like shading, sticker-pack composition, clean transparent background, no cluttered microtext.

negative constraints: no official logo imitation, no strawberry-hat logo, no anime human character, no photorealism, no thin lines, no illegible text, no extra words beyond `kali`.
